# Version 1: The Fool's Inquiry

2♦
: Weak Major (6-card, 5-10 HCP)
: NV 1st/3rd position: Can be 5-card suit

## Responses

2♥
: Relay (Pass/Correct), for all weak hands
  - Opener with Hearts: Pass
  - Opener with Spades: Bid 2♠

2♠
: **The Fool's Inquiry** (Artificial, INV+)
  - 2NT : Minimum with **Hearts**
  - 3♣ : Minimum with **Spades**
  - 3♦ : Maximum with **Hearts**
  - 3♥ : Maximum with **Spades**
  - 3NT : Solid Suit (AKQxxx)

2NT
: **Puppet to 3♣**
  - For weak minor signoffs (pass 3♣ or correct to 3♦)
  - Or Strong Balanced (Rebid 3NT)

3♣/♦
: Natural, GF

3M
: Preemptive (to play if it matches, pass/correct otherwise)

4♣
: Transfer to ♥
  - Opener bids 4♥

4♦
: Transfer to ♠
  - Opener bids 4♠

# Version 2: Three-Range Multi (Problems.md style)

2♦
: Weak Major (Three ranges: 5-6, 7-8, 9-10)

## Responses

2♥
: Relay (Pass/Correct)

2♠
: **Inquiry 1** (Suit + Range Ask)
  - 2NT : **Hearts**, Low Range (5-6)
  - 3♣ : **Spades**, Low Range (5-6)
  - 3♦ : **Hearts**, Medium Range (7-8)
  - 3♥ : **Spades**, Medium Range (7-8)
  - 3♠ : **Hearts**, High Range (9-10)
  - 3NT : **Spades**, High Range (9-10)

2NT
: **Inquiry 2** (General Feature Ask / Strong)

3m
: **Sign-off** with own minor

3M
: **Invitational** with other Major (Pass/Correct)
  - Opener passes if suit matches, converts if not (and likely passes the converted bid).

4m
: **Game Forcing** with own minor

4M
: **Game Forcing** with other Major (Pass/Correct to game)
